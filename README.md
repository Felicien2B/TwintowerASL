# TwintowerASL
Auto splitter for A Game, Creating twin towers with balancing (Windows)

- Starts when the first selected level appears.
- Splits at selected levels (1 to 10, and "secret" levels: 15, 20, 25, 30, 35, 40).
- You can enable transitions to get Individual Level times.
- Resets when you go back to the title screen.

Tested on Windows 10/11.
